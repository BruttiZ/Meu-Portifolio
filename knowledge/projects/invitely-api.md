A Invitely API é uma API modular em Go para autenticação, eventos, convidados,
RSVP, dashboard, analytics, orçamento, presentes e lembretes por e-mail.

Utiliza Gin, PostgreSQL com pgx, Supabase Auth, Docker e Swagger. As rotas
protegidas usam Bearer token. O projeto registra a evolução de Victor em Go,
modularidade e desenho de APIs.
